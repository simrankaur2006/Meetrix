# Meetrix 🎥

**Meetrix** is a real-time video conferencing web application inspired by Zoom, enabling users to create or join secure meeting rooms for seamless online communication. It offers high-quality video and audio calling along with essential collaboration features for an interactive meeting experience.

## 🚀 Features

- 🎥 Real-time Video Calling
- 🎤 Audio Mute/Unmute
- 📹 Camera On/Off
- 💬 Live Chat During Meetings
- 👥 Create & Join Meeting Rooms
- 🔗 Unique Meeting Links
- 🖥️ Screen Sharing
- 📱 Responsive UI
- 🔒 Secure Peer-to-Peer Communication
- 🚪 Leave Meeting Anytime

## ⚙️ Installation

### Clone the Repository

```bash
git clone 
cd meetrix
```

### Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd ../server
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

The application will run at:

```
Frontend: http://localhost:3000
Backend: http://localhost:5000
```
